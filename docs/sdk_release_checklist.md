# SDK Stable Release Checklist

- [ ] Exemplo mínimo funcional publicado com `include/` + runtime do target (`nova3d.dll`/`.a`/`.so`).
- [ ] Validação de import concluída para 2–3 formatos principais (OBJ, FBX, glTF).
- [ ] Documentação de link e runtime revisada por plataforma (Windows/Linux/Android).
- [ ] Third-party notices e licenças consolidadas no pacote da release.
- [ ] Versionamento semântico atualizado e política ABI revisada.
- [ ] `docs/build_matrix.md` atualizado com status real da release.
- [ ] Smoke test executado no target principal da release.
